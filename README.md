# ☀️ Weather App

A simple web app that fetches real-time weather data for any city using the OpenWeather API. Built with HTML, CSS, and JavaScript.

## 🚀 Features

* Search weather by city
* Displays temperature, wind speed, humidity, and weather condition
* Card-style UI for clear display
* Real-time API data

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript
* OpenWeather API

## ▶️ How to Run

1. Clone or download the project
2. Open `index.html` in your browser

## 🔑 API Setup

* Get a free API key from [OpenWeather](https://openweathermap.org/api)
* Replace the API key in your JavaScript file

## 🎮 How to Use

* Enter a city name in the search bar
* Click search
* View weather details in the card
